Contributing

How to contribute to LIME Toolbox

Please note we have a code of conduct, please follow it in all your interactions with the project.

Did you find a bug?

  • Ensure the bug was not already reported by searching on GitHub under Issues.

  • If you’re unable to find an open issue addressing the problem, open a new one. Be sure to include a title and clear description, as much relevant information as possible (pictures, log files, etc.), and a step by step description so anyone can replicate the problem.

Do you have questions about the source code?

  • Ask any question about how to use the LIME Toolbox on the LIME forum at lime.uva.es/forums
