The lime toolbox performs simulations of lunar irradiance, reflectance and polarization for a given point and datetime. It also performs comparisons for some given observations files in GLOD format.

It won't work unless given only one of the options (-h|-e|-l|-s|-c).

Options:
  -h, --help		 Displays the help message.
  -e, --earth		 Performs simulations from a geographic point.
			 -e lat_deg,lon_deg,height_m,%Y-%m-%dT%H:%M:%S
  -l, --lunar		 Performs a simulation from a selenographic point.
			 -l distance_sun_moon,distance_observer_moon,selen_obs_lat,selen_obs_lon,selen_sun_lon,moon_phase_angle
  -s, --satellite	 Performs simulations from a satellite point.
			 -s sat_name,%Y-%m-%dT%H:%M:%S
  -c, --comparison	 Performs comparisons from observations files in GLOD format.
			 -c "input_glod1.nc input_glod2.nc ..."
  -o, --output		 Select the output path and format.
			 If it's a simulation:
			   GRAPH: -o graph,(pdf|jpg|png|svg),refl,irr,polar
			   CSV: -o csv,refl.csv,irr.csv,polar.csv
			   LGLOD (netcdf): -o nc,output_lglod.nc
			 If it's a comparison:
			   GRAPH: -o graph,(pdf|jpg|png|svg),(DT|MPA|BOTH),comparison_channel1,comparison_channel2,...
			   CSV: -o csv,(DT|MPA|BOTH),comparison_channel1.csv,comparison_channel2.csv,...
			   GRAPH directory: -o graphd,(pdf|jpg|png|svg),(DT|MPA|BOTH),comparison_folder
			   CSV directory: -o csvd,(DT|MPA|BOTH),comparison_folder
			   LGLOD (netcdf): -o nc,output_lglod.nc
  -f, --srf		 Select the file that contains the Spectral Response Function in GLOD format.
  -t, --timeseries	 Select a CSV file with multiple datetimes instead of inputing directly only one datetime. Valid only if the main option is -e or -s.
